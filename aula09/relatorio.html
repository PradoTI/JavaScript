Aqui está um resumo sobre o que aprendi hoje:

Hoje aprendi um pouco sobre como usar operadores lógicos ( !, && e || ), os operadores ternários ( ? e : ) e para fechar um pouco sobre DOM.

Sendo ( ! == nao ) para negação, ou seja : quando for false == true e vice versa.

( && == e ) para conjunção, ou seja : se apontar todas as condições como true então será true mesmo, senão será false.

Ex:
var a = 2
var b = 4
a > b (false) && b > a (true) == false, apenas uma apontou true
a < b (true) && a > b - 3 (true) == true, as duas apontaram true

( || == ou ) para disjunção, ou seja : se uma das condições estiver verdadeira (true) então apontará como true.

Ex:
var a = 2
var b = 4
a > b (false) || b > a (true) == true, uma apontou como verdadeira
a > b (false) || a < b - 3 (false) == false, nenhuma apontou como verdadeira

Arvore DOM :

- A janela do navegador se chama 'window' que é um objeto DOM

Na arvore, quem vem antes é 'parent' quem vem depois é 'child' .
